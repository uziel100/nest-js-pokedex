## Ejecutar en desarrollo

1. Clonar repositorio

## Installation

```bash
$ pnpm install

npm i -g @nestjs/cli

```
## Up Database
```
docker compose-up -d
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
